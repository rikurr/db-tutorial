## Next.js Hasura TypeScript環境構築


## フロントエンドのローカル環境
yarn web dev

## ローカルのHasura起動
yarn hasura start

## Hasuraコンソール
yarn hasura console 


## フロントエンドテスト
yarn web test
## フロントエンドcodegen生成
yarn web codegen

### Stack
- Apollo GraphQL
- Typscript
- Minireset.css
- Hasura
- Css module
- Eslint, Prettier
- Jest
- Story Book
